# Developer Logos

Place developer logo images in this folder using the following naming convention:

## Naming Convention
Use lowercase with hyphens for multi-word names:
- `arada.png`
- `nshama.png`
- `iman.png`
- `azizi.png`
- `binghatti.png`
- `baya.png`
- `damac.png`
- `omniyat.png`
- `ellington.png`
- `emaar.png`
- `meraas.png`
- `nakheel.png`
- `object-1.png`
- `samana.png`
- `select-group.png`
- `sobha.png`
- `aldar.png`
- `al-habtoor.png`
- `al-futtaim.png`
- `deyaar.png`
- `dubx.png`
- `mag.png`
- `union-properties.png`
- `danube.png`

## Image Requirements
- Format: PNG (preferred) or JPG
- Recommended size: 200x200px to 400x400px
- Background: Transparent (for PNG) or white
- Aspect ratio: Square or close to square

## Note
If a logo is not found, the system will automatically display the developer's initials as a fallback.
